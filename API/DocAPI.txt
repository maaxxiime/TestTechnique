Une fois avoir cloné le repository, se rendre dans le terminal de commande intégré à Visual Studio Code,
effectuer la commande "npm install" dans le dossier "API" et faire la même chose dans le dossier "react-app".

Pour tester l'API, il manque le fichier .env qui n'est pas accessible dans le repository vu qu'il s'agit de données sensibles.
Il contient seulement deux paramètres :

DB_ACCESS = " lien de la collection de mongoDB "
TOKEN_KEY = " Une clé de token créé sur un site spécialisé "
